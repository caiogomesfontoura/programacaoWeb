Introdução

Programação web é o processo de criar páginas e aplicações para o navegador.

  • HTML: estrutura do conteúdo.
  • CSS: aparência e organização visual.
  • JavaScript: comportamento e interatividade.

HTML

HTML usa tags para organizar títulos, textos, listas, links e outras partes da página.

<h1>Titulo principal</h1>
<p>Paragrafo de exemplo.</p>
<a href="https://developer.mozilla.org">Ler mais</a>

CSS

CSS define estilo com regras simples de seletor e propriedades.

body {
    font-family: Arial, sans-serif;
    color: #1f2937;
}

h1 {
    font-size: 2rem;
}

Layout: Flexbox e Grid

Flexbox é ideal para layouts de uma dimensão (linhas ou colunas):

.container {
    display: flex;
    gap: 1rem;
    justify-content: space-between;
    align-items: center;
}

CSS Grid é perfeito para layouts bidimensionais:

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1rem;
}
